# scala-native-prac

A practice of [scala-native](http://www.scala-native.org/en/latest/).

## Run with `sbt run`

```bash
sbt run
```

## Run with binary

```bash
sbt nativeLink
./target/scala-2.11/scala-native-prac-out
```

`./target/scala-2.11/scala-native-prac-out ` is a binary file!
